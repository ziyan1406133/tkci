Cara penggunaan aplikasi di server lokal :

- Pastikan sudah memiliki aplikasi XAMPP dengan
  versi php minimal 7.3
- Extract tkci.rar di folder xampp/htdocs
- Buka xampp control panel
- Jalankan apache dan Mysql
- Buka cmd
- Navigasi ke folder tkci,
  contoh :
  ~ d:
  ~ cd xampp/htdocs/apriori-frontend

- Jalankan perintah berikut :
  ~ php artisan serve

- Sistem akan memberi tahu url webapp
  contoh : http://localhost:5000 atau http://localhost:8000

- Akses url tersebut untuk membuka aplikasi dari browser

- Jangan tutup cmd sampai penggunaan aplikasi selesai
